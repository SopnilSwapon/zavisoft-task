# Zavisoft Frontend Task

A responsive e-commerce frontend built with Next.js based on the provided Figma design.  
This project integrates the Platzi Fake Store API to display products, categories, and product details.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Axios
- Context API (Cart state)
- Swiper.js

---

## Features

- Product listing from API
- Product detail page
- Category slider
- Responsive UI (mobile & desktop)
- Loading, Error & Empty states
- Local cart state (Context API)
- Skeleton loaders

---

## API Used

- Platzi Fake Store API (Products, Get product by id, categories)

---

## Setup Instructions

```bash
git clone https://github.com/SopnilSwapon/zavisoft-task.git
cd zavisoft-task
npm install
npm run dev
```
